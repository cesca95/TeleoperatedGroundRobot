Detailed Description

Leap Motion Teleoperation.

Date
19-02-2019
Authors
Nicolas Dejon and Adam Berka

Script that sends the raw yaw, pitch and roll angles in degrees to the /orientation topic.
It also converts the leap motion data into Twist messages by a defined hand-gesture in order to teleoperate the simulated robot turtlesim
