\doxysubsection{Detailed Description}
Leap Motion Teleoperation. 

\begin{DoxyDate}{Date}
19-\/02-\/2019 
\end{DoxyDate}
\begin{DoxyAuthor}{Authors}
Nicolas Dejon and Adam Berka
\end{DoxyAuthor}
Script that sends the raw yaw, pitch and roll angles in degrees to the /orientation topic.~\newline
 It also converts the leap motion data into Twist messages by a defined hand-\/gesture in order to teleoperate the simulated robot turtlesim
